Найдите номер модели, скорость и размер жесткого диска для всех ПК стоимостью менее 500 дол. Вывести: model, speed и hd

Задание: 6 
Для каждого производителя, выпускающего ПК-блокноты c объёмом жесткого диска не менее 10 Гбайт, найти скорости таких ПК-блокнотов. Вывод: производитель, скорость.
РЕШЕНИЕ:
SELECT DISTINCT maker, speed FROM laptop INNER JOIN (
SELECT * FROM product
WHERE type = 'laptop')product On laptop.model = product.model
WHERE hd>=10;
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Задание: 7
Найдите номера моделей и цены всех имеющихся в продаже продуктов (любого типа) производителя B (латинская буква).
РЕШЕНИЕ:
SELECT DISTINCT pc.model, price FROM pc JOIN Product ON pc.model = Product.model WHERE maker LIKE 'B%'
UNION
SELECT DISTINCT laptop.model, price FROM laptop JOIN Product ON laptop.model = Product.model WHERE maker LIKE 'B%'
UNION
SELECT DISTINCT printer.model, price FROM printer JOIN Product ON printer.model = Product.model WHERE maker LIKE 'B%';
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Задание: 8 
Найдите производителя, выпускающего ПК, но не ПК-блокноты.
РЕШЕНИЕ:
SELECT DISTINCT maker FROM Product
WHERE type = 'pc' AND maker NOT IN (SELECT maker FROM Product WHERE type = 'laptop');

Найдите модели принтеров, имеющих самую высокую цену. Вывести: model, price
РЕШЕНИЕ:

FROM Printer

Задание: 14
Найдите класс, имя и страну для кораблей из таблицы Ships, имеющих не менее 10 орудий.
РЕШЕНИЕ:
SELECT  Ships.class, name, country FROM Ships INNER JOIN Classes On Ships.class = Classes.class
WHERE numGuns>=10;
